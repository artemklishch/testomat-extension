# testomat-extension

1. Create an account in the Testomat.io
2. Create the project
3. Take the "Project Reporting API key" value from the project settings? create the ".env" file in the project root and assign the value to this variable
4. Run installing Maven dependencies
5. Run the file "WatcherTest" - you must see in the "Runs" tab in the Testomat.io platform the message about tests and their number.
